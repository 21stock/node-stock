Date()

Various date methods useful for stock trading.

Yes, I know that it is not good to augment built-in types, but I have considered all pros and cons and decided to pick this solution anyway.

A daystamp is a date format in form of "YYYYMMDD". It's used through all the node-stock package. It could be specified both in string or integer and all the methods and functions must accept either type (mind .toString() and parseInt() for easy implementation).

Date.parseDaystamp(daystamp) static

Parse daystamp into Date. It's a factory method.

  • daystamp — daystamp to parse

Returns:

  • Date date parsed from daystamp, time set to 00:00:00

daystamp()

Return daystamp representation of a Date.

Returns:

  • String daystamp

Date.parseUnixtime(unixtime) static

Parse unixtime into Date. It's a factory method.

  • Integer unixtime — unixtime to parse.

Returns:

  • Date parsed date object with correct time set.

unixtime()

Return unixtime representation of a Date.

Returns:

  • Integer unixtime.

Date.unixtime() static

Return current unixtime.

Returns:

  • Integer current unixtime.

Date.fillEmptyDays(from, to) static

Return an array of Dates (days) between two dates, inclusive.

  • Date from — starting day
  • Date to — ending day

Returns:

  • Array array of Dates between those.

If from and to are the same, it will only return one element.

Example: 

var from = new Date('2011-06-15');
var to   = new Date('2011-06-18');
var days = Date.fillEmptyDays(from, to);
// days == [
	new Date('2011-06-15'), 
	new Date('2011-06-16'), 
	new Date('2011-06-17'), 
	new Date('2011-06-18')
];

Note: this function is not time-safe, so please use only for dates.

getDayName()

Get short human readable day name for given Date, like "Sun", "Mon", ...

Returns:

  • String day name.

setCurrentDayMinute(minute)

Set given Date's time to a day minute specified. A day minute is zero-based number of minute in a day, e.g. 570 for 9:30.

  • Integer minute — day minute to set as time.

getCurrentDayMinute()

Get current day minute of a given Date. A day minute is zero-based number of minute in a day, e.g. 570 for 9:30.

Returns:

  • Integer day minute

Date.minuteToFormat(minute, format) static

A helper function. Format given day minute according to format (see date-utils format specifiers).

  • Integer minute — day minute
  • String format — date-utils format specifier

Returns:

  • String formatted time

Example: 

Date.minuteToFormat(571, 'H24:MI:SS'); // "9:31"
