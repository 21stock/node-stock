Symbol

Represents one particular symbol storage in the ticks database. Essentially this is just a wrapper around a list of *.ticks files with handy access methods.

  • String dbPath — path to ticks database.
  • String symbol — symbol name, must be uppercase.

Tick files list is generated from dbPath/symbol/*.ticks.

Example: 

var symbol = new Symbol('/home/tickers', 'AAPL');
if (!symbol.load()) {
	console.log("Oops?");
	return;
}

var daystamp;
while ((daystamp=symbol.next())) {
	console.log("Day: %s", daystamp);
}

setIsActive(isActive)

Set active flag for the ticker. This flag is used by downloader to skip download this ticker from IQFeed if the flag is set. Defaults to false, of course. Don't forget to call .save() after setting the flag.

isActive()

Get active flag for the ticker.

save()

Save updated information for the ticker. Now this is just the meta information that contains active flag.

load()

Load the ticker from dbpath. Essentially does load the list of all days available for the ticker and the meta information.

Returns:

  • Boolean if loaded successfully.

forEachDay(cb, context)

Single-step iterator over all days.

  • Function cb — callback.
  • context — set this to this.

Callback function will receive one argument: the daystamp, string.

rewind()

Rewind current iterator position to the beginning.

nextDay()

Get the next day from list of days.

Returns:

  • String daystamp.

firstDay()

Get the first day from list of days.

Returns:

  • String daystamp.

lastDay()

Get the last day from list of days.

Returns:

  • String daystamp.

count()

Get the count of available days.

Returns:

  • String count of days.

dayExists(daystamp)

Check if this daystamp exists for this ticker.

  • String daystamp — day to look for.

Returns:

  • Boolean
